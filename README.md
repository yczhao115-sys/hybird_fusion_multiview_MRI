# hybird_fusion_multiview_MRI

This repository provides the official PyTorch implementation of the core model architecture proposed in:

Hybrid Multi-View MRI Fusion for Clinically Significant Prostate Cancer Diagnosis via Intra- and Inter-View Transformers

The repository currently contains model-level code only, focusing on the architectural design of the proposed hybrid fusion framework.
Training and test code will be released in a future update.
